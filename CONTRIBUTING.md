# Contributing

This repository is the canonical intellectual property of Paul Roth and RothMethods. It is not an open-source project.

---

## Who Can Contribute

Contributions to this repository are by invitation only. All changes require explicit authorization from Paul Roth.

---

## Authorized Changes

Authorized contributors may:

- Add new Method documentation as Methods are developed and released
- Update CHANGELOG.md to reflect new versions
- Correct typographical errors in canonical documents
- Add new outcome documentation as cases are cleared for publication

---

## Unauthorized Changes

The following are not permitted without explicit written authorization:

- Modifying canonical framework definitions
- Altering proprietary language or signature lines
- Restructuring the repository architecture
- Adding content that is not drawn from confirmed RothMethods source material

---

## Process

All proposed changes must be submitted for review by Paul Roth before committing. No change to canonical content is recognized as authoritative until confirmed by the repository owner.
